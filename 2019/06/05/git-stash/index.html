stash-暂存区

查看暂存

1、git stash list

2、git stash show [stash]

3、git stash drop [stash]

4、git stash clear

从workspace添加到stash

1、git stash save [msg]

猜想：stash是在队列头部插入数据

将stash还原到workspace

1、git stash apply [stash]

2、git stash pop

将stash 直接提交到local

git stash branch [stash]
