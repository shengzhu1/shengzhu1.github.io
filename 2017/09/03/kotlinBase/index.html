运算符

以下是位运算符的完整列表(只适用于 Int 类型和 Long 类型):

  • shl(bits) – 带符号左移 (等于 Java 的<<)
  • shr(bits) – 带符号右移 (等于 Java 的 >>)
  • ushr(bits) – 无符号右移 (等于 Java 的 >>>)
  • and(bits) – 按位与(and)
  • or(bits) – 按位或(or)
  • xor(bits) – 按位异或(xor)
  • inv() – 按位取反
